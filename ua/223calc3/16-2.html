Calculus III

§ 16.2: Line Integrals

There are some new concepts here, but the calculation is actually much more straightforward than it might appear at first glance. You will usually be supplied with a parametric description of a path; in particularly simple cases, such as a line segment or an arc of a circle, you may need to construct a parametrization yourself using the techniques of § 12.5 and/or Chapter 13. In any case, once you have a parametrization, you need only express everything in terms of the parameter and integrate. Especially if you have good habits with handling differentials, the notation basically takes care of itself. (If you don’t have good notational habits, start being more careful about that—better late than never!)

Line integrals of scalar fields

The first few subsections deal with line integrals of scalar fields—i.e., real-valued functions. The default is integration with respect to arc length, ds. Recall from Chapter 13 that the differential arc length ds = |r′(t)| dt, which can be expanded to various radical expressions.

If the integrand is simply 1, you have the arc length calculation from Chapter 13. As long as the integrand is nonnegative, we have several options for interpretation: If it’s a height (z-value), the integral represents an area of a particular sheet, as illustrated in Figure 2. If the function is the linear mass density of a wire, the integral along the curve describing the wire represents total mass; see Example 3 and Figure 6. I work through a quick example in “§ 16.2a: Exercise #2” (YouTube | Panopto).

Integrals with respect to a particular variable are slightly harder to conceptualize, but somewhat easier to evaluate, since instead of the relatively complex ds, you have a much simpler differential like dx, dy, or dz. Recall from the definition of differentials (§ 3.10) that if x is a function of t, then dx = x′(t) dt, and similarly for other variables. The text presents messy, explicit formulae (box 7), as it tends to do, but if you’re well practiced working with differentials, you have nothing new to memorize here—just substitute everything in where it goes, as specified by the notation itself. I illustrate this method of computation in “§ 16.2b: Exercise #8” (YouTube | Panopto).

Line integrals of vector fields

There are several ways of thinking about line integrals of vector fields; before I get into the geometry and calculation, I want to address the way the textbook leans heavily on its application to computing the work done by a force, because that’s probably one of the most common applications. If you have some intuition for work and force from past studies of physics, thinking of these line integrals in those terms can be helpful. However, if you do not, that’s fine; this is not a physics course. Feel free to gloss over discussions of work, and just take “compute the work done by this force in moving a particle along this path” to be code for “compute the line integral of this vector field along this curve”.

Now then. We typically write line integrals of vector fields in a form like ∫C F⋅dr. This can be interpreted in a few different ways, each with its uses.

Definition 13 and the preceding discussion show that the integral is equal to ∫C F⋅T ds. This is not generally useful for computation, but it does illustrate the geometric meaning of the line integral: The integrand, F⋅T, is a dot product, so it’s scalar-valued; hence, this is an integral of the type we saw in the earlier portion of this section. Recall that T is a unit vector, so F⋅T is the component of F in the direction of T. (If needed, review projections from § 12.3.) Furthermore, since T is tangent to the curve and points in the direction of movement, this expression tells us the portion of F that points in the same direction; in a sense, it measures the degree to which F “agrees with” motion along the path C. A negative value indicates that it’s pushing backwards against the motion described. At each point along the curve, then, F⋅T measures how much the vector field moves with or against the curve. Summing (integrating) all of these along the length (ds) of the curve, we get the total or net amount of this “agreement”. If positive, the field points along with the path more than it points against it; if negative, the reverse. If zero, these are perfectly balanced.

So that tells us what this means, but like I said, we don’t usually use that form for computation. For that, we work from dr. One way to interpret this is like any other differential, so dr = r′(t) dt = ⟨x′(t), y′(t), z′(t)⟩ dt. Alternatively, consider that dr = ⟨dx, dy, dz⟩ = ⟨x′(t) dt, y′(t) dt, z′(t) dt⟩. Observe that these are equal. Observe also that, again, if you just use the symbols and evaluate the expressions and differentials as stated, writing everything in terms of the parameter t, the notation takes care of things for you, and you don’t have anything to memorize! On the other hand, if the expressions that fall out of this are more complex and unwieldy than you would like, you can use the “expanded” versions shown at the end of the section, after Example 8. Some students find it much easier to separately write out and simplify P, Q, and R, and dx, dy, and dz, then combine them together afterward to integrate—or perhaps even integrate them separately! I illustrate both bookkeeping methods in “§ 16.2c: Exercise #22” (YouTube | Panopto); you can decide which you prefer, or find some combination or alternate technique that works better for you.

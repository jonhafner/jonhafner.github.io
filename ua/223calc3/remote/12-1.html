Surfaces and Solids

Read this subsection. The opening paragraph touches on a point that is much more significant than it might seem at first, which I’ll state in a moment. If the following gets too abstract to follow, you can just focus on the examples in the text and then move on to the next subsection, but if this does make sense to you, I think it will be enlightening.

The broader point is this: In general, the solution set of an equation in n real variables is usually an (n – 1)-dimensional object, which divides the space ℝn into two or more distinct regions.

For example, if n = 1, we have a single real variable—say, x—in the one-dimensional space ℝ, the real line. Think of any ordinary, algebraic equation in x; if it has any solutions, these will usually be isolated points. (For example, a linear equation, such as 2x + 3 = 7, will have a single solution.) Each such point is a zero-dimensional object, and when we plot it on the real line, it splits ℝ into two intervals, one to the left and one to the right.

If n = 2, we have a pair of real variables—say, (x, y)—representing a location in the two-dimensional space ℝ2, the Euclidean real plane. Consider some common equations in x and y; a few simple examples include 2x + 5y = 10, x = y2, y = sin x, or x2 + y2 = 9. The graphs of their solution sets will often be lines or curves. Though they might be bent to occupy a two-dimensional space, these are in some sense one-dimensional objects, since you can locate any point on the curve by specifying a directed distance from a given reference point. (Think about the arc length function from Calc II.) Furthermore, many such graphs divide the plane into two regions—a top and a bottom, or a left and a right, or an inside and an outside.

Similarly, as the text casually remarks in passing, an equation in three variables will often have a solution set of (x, y, z) triples forming a two-dimensional surface that divides the three-dimensional space ℝ3 into two or more regions.

This pattern continues into higher-dimensional spaces, though such geometries can be rather challenging to visualize!

(Note my use of words like “many” and “often”. The statements above are common with simple examples, but there are plenty of exceptions. For example, in the n = 2 setting, a hyperbola cuts the plane into three regions, the tangent function cuts it into infinitely many regions, and the floor function, a.k.a. the greatest integer function, doesn’t divide the plane into separate pieces at all! But note that even these are built out of curves, line segments, etc.; the general idea still holds, that graphs of equations tend to drop down one dimension from the containing space.)
